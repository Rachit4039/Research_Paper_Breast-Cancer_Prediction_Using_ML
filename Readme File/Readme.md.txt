#  Breast Cancer Classification using Machine Learning

## 📌 Overview

This project applies machine learning techniques to classify breast cancer tumors as **malignant (0)** or **benign (1)**.
We compare multiple models to determine which performs best on the dataset.

---

## 🎯 Objectives

* Analyze and preprocess the dataset
* Train multiple ML models
* Evaluate performance using standard metrics
* Compare models and draw conclusions

---

## 📊 Dataset

* **Breast Cancer Wisconsin Dataset**
* Loaded using:

  ```python
  from sklearn.datasets import load_breast_cancer
  ```
* No external file required

---

## 🤖 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)

---

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix
* ROC Curve
